## Cross-cutting patterns (apply to every fix below)

1. **Decks render exactly the outline directives — nothing more.** No deck adds visuals beyond
   its `.md` outline, so every gap must be fixed in the outline first, then rebuilt with
   `build-lecture-deck`.
2. **Colour intent is almost never stated in directives** (systemic, L01–L07). Deck builders
   guessed — mostly correctly, once wrongly (see the L0505 bug). When touching any directive,
   fold in one phrase: "cyan = …, coral = …".
3. **Quantitative/cost claims lack bar charts** — the flavor FRONTEND-STYLE says cost lessons
   lean on hardest. L01 §7–8, L02 4.4, L06 5.1 (a cost-claim lesson with *no bar chart at all*),
   L07 4.2.
4. **Motifs stop one or two beats early.** Every lesson already owns a recurring visual (L01
   distribution chart + window bar; L02 pill-stack + validator gate; L03 `extract_node` box;
   L04 the chain; L05 the router; L06 the two-prompt contrast + critique loop; L07 the
   four-message strip + three-outcome fan) that isn't reused on the slides where "same picture,
   one thing changed" would teach hardest.
5. **Notebooks print numbers instead of plotting.** L01's six lecture notebooks contain zero
   plots despite LECTURES.md naming live plotting as the main reason a lecture becomes a
   notebook.
