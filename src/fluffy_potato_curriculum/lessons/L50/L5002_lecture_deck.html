Reuse over re-derive

  • The loop, the tracing, the eval types all already live in the shared common/ layer. You import them — you don't rebuild them.

The one tool, and a real two-tool decision

  • find_matching_record normalizes any format to (vendor, amount) and looks it up — a warranted tool, because reliable cross-format matching is not eyeball work.

Dim inputs, two cyan tools — the choice between them is the decision worth tracing.

Slide 5.1

“It ran once” is not done

  • One green run of a non-deterministic agent proves little. The habit the whole mini course built — trace before you guess, eval or it's vibes — is what makes a slice finished.
  • The one line to carry out: the slice is done when it's traced and has one case that would catch its regression.
