Slide 1.1

A bulleted content slide

  • The default body is a ul.points list — cyan ring bullets, one idea per line, kept tight so nothing overflows the stage.
  • Inline emphasis: bold ink for the key term, code for identifiers, and a coral warn span for the failure to avoid.
  • Aim for around four points; reading-first density means self-contained lines, not a wall of text.
  • Reveal animation staggers each point in as the slide becomes .visible.

Use a callout for the one load-bearing sentence a slide is built around.
